/* from/to: atalho pra 0% e 100% — bom pra animações simples */
@keyframes mover {
    from { transform: translateX(-40px); }  /* ponto de início */
    to   { transform: translateX( 40px); }  /* ponto de fim */
}

/* porcentagens: controla cada etapa do meio */
@keyframes etapas {
    0%   { transform: translateX(-40px); background: green; }
    25%  { transform: translateY(-40px); background: yellow;}
    50%  { transform: translateX( 40px); background: red;   }
    100% { transform: translateX(-40px); background: green; }
}

/* 0% e 100% iguais = loop sem "salto" */
@keyframes pulso {
    0%, 100% { transform: scale(1);   }
    50%      { transform: scale(1.5); }
}

Exemplos de animacao

fade in

01. Fade In

Entrada suave para apresentar conteudo sem susto.

Essa animacao funciona quando voce quer que um elemento apareca com elegancia. Muito usada em cards, blocos de texto e interfaces que precisam passar sensacao de cuidado.

Quando usar: introducao de secoes, listas e paineis.

02. Slide Up

Movimento que guia o olhar e cria direcao.

O slide da sensacao de progresso. Ele e forte para apresentar etapas, beneficios ou qualquer bloco que precise parecer que esta chegando no momento certo.

Quando usar: timeline, onboarding e chamadas de destaque.

Texto que entra como revelacao quase cinematografica

03. Text Reveal

Palavras aparecendo em camadas para valorizar a mensagem.

Em vez de jogar o texto inteiro de uma vez, essa tecnica monta ritmo. Ela funciona muito bem quando a frase importa e precisa chegar com mais peso emocional.

Quando usar: frases de impacto, manifestos e headlines.

04. Stagger

Itens entrando em sequencia para organizar informacao.

O stagger e um classico porque resolve bem listas, menus e galerias. A aparicao em cadeia deixa a leitura mais leve e ajuda a pessoa a entender a ordem do conteudo.

Quando usar: cards, beneficios, listas de recursos e menus.

05. Parallax

Profundidade visual para deixar a pagina menos estatica.

Quando camadas se movem em velocidades diferentes, o site ganha sensacao espacial. O segredo e usar com controle, para enriquecer a experiencia sem cansar.

Quando usar: hero section, banners e storytelling visual.

06. Floating Elements

Micro movimento continuo para manter a interface viva.

Pequenos objetos flutuando passam sensacao de suavidade e sofisticacao. E uma camada boa para fundos, areas conceituais e composicoes que precisam de ar.

Quando usar: fundos decorativos, sessoes premium e cenas abstratas.

/* animation-delay: espera antes de começar */
animation-delay: 0s;      /* começa imediatamente (padrão) */
animation-delay: 0.5s;    /* espera meio segundo */
animation-delay: -0.75s;  /* delay negativo: começa no meio da animação!
                             útil pra criar defasagem em elementos iguais */

/* cascata: delay crescente em cada elemento */
.item-1 { animation-delay: 0s;   }
.item-2 { animation-delay: 0.2s; }
.item-3 { animation-delay: 0.4s; }

/* animation-iteration-count: quantas vezes repete */
animation-iteration-count: 1;         /* 1 vez e para (padrão) */
animation-iteration-count: 3;         /* exatamente 3 vezes */
animation-iteration-count: 2.5;       /* aceita decimal! */
animation-iteration-count: infinite;  /* repete pra sempre */

/* animation-direction: pra onde a animação vai */
animation-direction: normal;            /* padrão: sempre de 0% → 100% */
animation-direction: reverse;           /* sempre de 100% → 0% */
animation-direction: alternate;         /* alterna: → ← → ← (sem salto!) */
animation-direction: alternate-reverse; /* alterna: ← → ← → */

/* animation-fill-mode: o que acontece antes/depois de rodar */
animation-fill-mode: none;        /* padrão: volta ao estado original */
animation-fill-mode: forwards;    /* FICA no estado final quando termina */
animation-fill-mode: backwards;   /* aplica o estado inicial durante o delay */
animation-fill-mode: both;        /* combina os dois — mais usado! */

/* exemplo clássico: aparece e FICA aparecido */
.aparecer {
    opacity: 0;
    animation: fadeIn 0.5s ease forwards;  /* sem forwards voltaria pra 0! */
}

/* animation-play-state: pausa ou retoma */
.spinner {
    animation: rodar 1s linear infinite;
    animation-play-state: running;   /* padrão: rodando */
}
.spinner:hover {
    animation-play-state: paused;    /* pausa ao passar o mouse */
}
/* via JS: elemento.style.animationPlayState = "paused" */

/* múltiplas animações: separa por vírgula */
.elemento {
    animation:
        vai-volta  2s  ease-in-out  infinite  alternate,  /* move */
        troca-cor  3s  linear       infinite  normal,     /* cor */
        pulsa      0.8s ease        infinite  alternate;  /* tamanho */
}
